# PRODUCT_SPEC.md

## 1. Product Name (working)
Drone Report Builder

## 2. Core Idea
A web-based tool that allows users to turn inspection images into structured, professional PDF reports quickly.

## 3. Target Users
- Drone operators doing inspections
- Small businesses (1–10 people)

## 4. Problem Statement
Capture → Analyze → Manually build report → Send PDF

Pain points:
- time-consuming
- repetitive formatting
- inconsistent layout
- unprofessional output

## 5. Solution
A structured report builder where layout is predefined and output is always clean.

## 6. Core Workflow
1. Create project
2. Upload images
3. Create findings
4. Annotate images
5. Add descriptions + data
6. Generate report
7. Export PDF

## 7. Report Structure
- Cover Page
- Summary Page
- Findings Overview
- Finding Detail Pages

## 8. Finding Structure
- image
- annotations
- severity (1–5)
- issue type
- description
- recommendation
- custom fields

## 9. Design Principles
- Structured over flexible
- Fast over customizable
- Consistency over creativity

## 10. Non-Goals
- No free design editor
- No template marketplace
- No collaboration (v1)

## 11. Value Proposition
Create a professional inspection report in minutes instead of hours.
